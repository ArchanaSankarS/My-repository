PROJECT: JobZ - MODERN JOB PORTAL
(Final Frontend Capstone Project)
JobZ is a high-performance React.js application designed to connect Job Seekers and Recruiters through a seamless, Single Page Experience.

PROJECT MOTIVATION :

Dual-Role Logic: Managing two different interfaces (Seeker vs. Recruiter) within one app.

Complex Workflow: Handling multi-step forms, resume uploads, and user sessions.

Real-World Architecture: Building a complete journey from job browsing to final application.



 TECHNICAL HIGHLIGHTS :

React Router v6: Smooth navigation without page reloads (SPA).

Role-Based Auth: Automatic redirection based on user login type.

Data Persistence: Uses LocalStorage to keep users logged in after a page refresh.

Dynamic UI: Used .map() to render 12+ tech jobs dynamically with live data.

Responsive Design: Fully optimized for Mobile, Tablet, and Desktop.



DESIGN & UI EXCELLENCE :

Shine Animation: Custom CSS @keyframes for premium, high-end text effects.

Interactive Toggle: A switchable UI in the Contact section between a Form and Google Maps.

Modern Layout: Engineered with CSS Grid and Flexbox for a sharp, clean look.



 KEY LEARNINGS :

Authentication: Successfully implementing Role-Based Access Control (RBAC).

Modular Code: Organizing the project into reusable React components for scalability.

Problem Solving: Managing complex navigation logic and persistent user states.
