This project , I have built a Multi-Page E-Services Portal that brings together different online services like Banking, Health, Learning, and E-Commerce into one single platform.

 To make navigation easy and professional, I used a Frame-based Layout, where the menu stays on the left side while the content changes on the right. 

Throughout this project, I learned how to use CSS Grid to organize service cards neatly and integrated video backgrounds to give the website a modern look. 

I also gained experience in building detailed User Registration Forms using semantic tags like fieldset and different input types to collect user data properly. 

By connecting multiple pages and managing their navigation, I have developed a strong understanding of how to structure a complex website and make it user-friendly.

 This project highlights my ability to combine HTML and CSS to create a functional and well-organized web application."
