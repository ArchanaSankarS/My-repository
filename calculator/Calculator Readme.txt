This project is a simple Calculator web application developed using HTML, CSS, and JavaScript. 

The main goal of this project was to understand how to build an interactive user interface and implement basic functionality using JavaScript.

In this project, I designed the calculator layout using HTML and styled it using CSS. 

I applied concepts like button styling, layout alignment, colors, shadows, and responsive positioning to create a clean and modern UI.

I also worked on styling input fields and creating a visually structured calculator design.

For functionality, I used JavaScript to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

I implemented features like clearing the display, deleting the last entered value, and displaying real-time input on the screen.

Through this project, I learned how to connect HTML, CSS, and JavaScript to build a working application.

It helped me understand DOM manipulation, event handling, and how user interactions work in web applications. 

This project strengthened my foundation in frontend development and improved my problem-solving skills.
